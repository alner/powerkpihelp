# Borders

“Borders” section of toolbar contains the following buttons:

![](../.gitbook/assets/image%20%2848%29.png)

* ![](../.gitbook/assets/image%20%283%29.png) -   insert borders on each side of the selected cell/cells
* ![](../.gitbook/assets/image%20%2816%29.png) -   insert border at the top of the selected cell/cells
* ![](../.gitbook/assets/image%20%28144%29.png) -   insert border at the bottom of the selected cell/cells
* ![](../.gitbook/assets/image%20%2855%29.png) -   insert left border for the selected cell/cells
* ![](../.gitbook/assets/image%20%28138%29.png) -   insert right border for the selected cell/cells
* ![](../.gitbook/assets/image%20%2895%29.png) -   remove borders on currently selected cell/cells
* ![](../.gitbook/assets/image%20%2825%29.png) -   change border line type and line width
* ![](../.gitbook/assets/image%20%28123%29.png) -   change border color
