# Borders

“Borders” section of toolbar contains the following buttons:

![](../.gitbook/assets/image%20%2834%29.png)

* ![](../.gitbook/assets/image%20%283%29.png) -   insert borders on each side of the selected cell/cells
* ![](../.gitbook/assets/image%20%2810%29.png) -   insert border at the top of the selected cell/cells
* ![](../.gitbook/assets/image%20%28108%29.png) -   insert border at the bottom of the selected cell/cells
* ![](../.gitbook/assets/image%20%2840%29.png) -   insert left border for the selected cell/cells
* ![](../.gitbook/assets/image%20%28103%29.png) -   insert right border for the selected cell/cells
* ![](../.gitbook/assets/image%20%2871%29.png) -   remove borders on currently selected cell/cells
* ![](../.gitbook/assets/image%20%2818%29.png) -   change border line type and line width
* ![](../.gitbook/assets/image%20%2893%29.png) -   change border color
